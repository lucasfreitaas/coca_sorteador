# Coca Dynamics Dashboard 🥤

Um sistema moderno e intuitivo para gerenciar a dinâmica de "cocas" (punições) da empresa. Este projeto substitui a gestão manual no banco de dados por um dashboard premium e automatizado.

## 🚀 Tecnologias

- **React + Vite**: Framework principal e bundler ultra-rápido.
- **Vanilla CSS**: Sistema de design customizado com CSS Variables e Glassmorphism.
- **Lucide React**: Biblioteca de ícones consistente.
- **Framer Motion**: Micro-interações e animações de interface.

## ✨ Funcionalidades (UI)

- [x] **Dashboard de Visão Geral**: Estatísticas rápidas de cocas pendentes e pagas.
- [x] **Fila de Cocas**: Visualização clara de quem deve, o motivo e a quantidade.
- [x] **Ações Rápidas**: Atalhos para adicionar novas cocas e realizar sorteios.
- [x] **Design Responsivo**: Adaptado para diferentes resoluções.
- [x] **Identidade Visual**: Integração com a logo da empresa e cores temáticas.

## 🛠️ Como Rodar

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
